iCook

Find recipes fast • Save favorites • Cook with confidence

Site Name

iCook — a short, memorable name that indicates the site's focus on cooking and recipes. Chosen for clarity, brandability, and mobile-friendly recognition.

Site Purpose

iCook is a lightweight recipe finder and organizer. Users can search for recipes, view clear step-by-step instructions, and save favorite recipes locally for quick access.

Scenarios

  1. Quick dinner: "What simple dish can I cook today?" The user wants to be able to find simple recipes with clear instructions for cooking.
  2. Save favorites: "I want to save and later view this lasagna recipe on my phone." The user expects a simple Favorites page that persists in the browser.

Color Scheme

The color palette below is used throughout this site plan. It will be used for headings, buttons, accents, and links.

--primary: #096B6B
Used for headings, primary buttons, and important accents.
--accent: #E76F51
Used for CTAs, links, and small badges.

Typography

Chosen fonts:

  • Headings: Georgia, system-ui, "Segoe UI", sans-serif
  • Body: Quicksand, system-ui, "Segoe UI", sans-serif — clear and compact for instructions and lists.

Key Pages & Features

  • Home: Featured recipes and a prominent search box.
  • Catalog: API-backed search.
  • Favorites: Saved recipes in localStorage with quick access to steps.
  • Recipe page: Ingredients, steps, prep/cook times, servings.
